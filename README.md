# EU | FRANCE & GERMANY | FOREIGN POLICY
Repository containing all code and documents connected to the Master's Thesis. Research question: "How do France and Germany's rhetorical positioning toward the Russia-Ukraine conflict, as expressed in press releases, align with or diverge from the EU's collective positioning, and how does this relationship evolve across key coordination events between 2014 and 2025?"

*data* directory contains all data, which is divided into *raw*, *translated*, *filtered*, and *final*. Data was processed and collected using the scripts present in *src* directory (see *scrapers*, *translators*, and *filters* directories )
