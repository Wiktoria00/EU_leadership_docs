# EU_leadership_docs
Repository containing all code and documents connected to the Master's Thesis on the Franco-German EU leadership and its constraints, in the context of foreign policy towards Russia in years 2014-2025.

*data* directory contains all data, which is divided into *raw*, *translated*, *filtered*, and *final*. Data was processed and collected using the scripts present in *src* directory (see *scrapers*, *translators*, and *filters* directories )
